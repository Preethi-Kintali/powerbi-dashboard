## 🔍 Overview

This project is an interactive **Power BI dashboard** designed to analyze chocolate sales, shipment performance, and supply chain insights.
It provides a comprehensive view of revenue, profit, shipment distribution, and product performance to support data-driven decision-making.

---

## 🚀 Key Metrics

* 💰 **Total Sales Amount**
* 📦 **Total Boxes Shipped**
* 🚚 **Shipment Count**
* 📈 **Total Profit**
* 📊 **Profit Percentage**

---

## 📌 Features

### 📊 1. KPI Overview

* Displays key business metrics at a glance
* Dynamically updates based on filters

---

### 📈 2. Sales Trend Analysis

* Year-over-Year comparison
* Tracks sales and shipment trends over time

---

### 📦 3. Shipment Distribution

* Visual representation of shipment volume
* Identifies distribution patterns

---

### 🏆 4. Top Products Analysis

* Highlights best-performing chocolate products
* Displays revenue contribution

---

### 👥 5. Salesperson Performance

* Top sales representatives ranked by:

  * Sales amount
  * Shipment volume
  * Profit percentage

---

### 🌍 6. Geographic Insights

* Sales distribution across regions/countries
* Helps identify high-performing markets

---

### 📋 7. Detailed Product Insights

* Product-wise breakdown:

  * Profit
  * Profit %
  * Shipments

---

## 🎛️ Interactivity

* 📅 Date range slicer
* 🌍 Region/Location filters
* 🔄 Cross-filtering across visuals
* 📊 Drill-down capability for deeper insights

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Data Modeling
* Excel/CSV Dataset

---

## 📁 Project Structure

* `Chocolate_Shipment_Dashboard.pbix` → Main Power BI dashboard
* `dataset.xlsx` → Source dataset

---

## ▶️ How to Use

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. If data source error occurs:

   * Go to **Transform Data → Data Source Settings**
   * Update dataset path
4. Use slicers to explore insights
